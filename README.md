# Reservia: A Front-End Project

Online Travel Agency Demo

## Introduction

This is an academic practice project aiming to replicate the [Reservia template](https://reservia.vercel.app "Reservia template"). It has nothing to do with and isn't endorsed by the original creator(s). I aim to recreate this template from scratch to enhance my HTML and CSS skills.

## Original Template

[Reservia](https://reservia.vercel.app "Reservia")

## Features to be implemented

- **Pixel-perfectness**: The webpage should be pixel-perfect resembling to the original template.
- **Animations**: The webpage should be further enhanced from its original version with the use of CSS animations.
